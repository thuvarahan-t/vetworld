VetWorld Product Store – Product Requirement Document (PRD)
1. Project Overview
Project Name: VetWorld Product Store
Client: VetWorld
Industry: Veterinary, Laboratory Equipment, Science Supplies

VetWorld requires a modern e‑commerce style product catalogue website where customers can browse products categorized into different scientific and veterinary equipment groups.

The system will allow users to:
• Browse products by category
• Filter products category‑wise
• Select product variants (types)
• Add items to cart
• Send order request via WhatsApp

The goal of the MVP is to create a simple ordering experience without a payment gateway where the order is sent directly through WhatsApp.
2. Technology Stack
Frontend
• React + TypeScript
• React Router
• TailwindCSS or Material UI

Backend
• Java 21
• Spring Boot
• Spring Security

When user clicks "Order via WhatsApp", the system generates a message containing:

Product Name
Selected Type
Quantity
Unit Price
Total Price
Overall Total

Example Message:

Hello VetWorld,
I would like to order the following items:

1. Test Tube
Type: 20ml
Quantity: 2
Price: Rs.180
Total: Rs.360

Total Order Price: Rs.360
Please confirm availability.
